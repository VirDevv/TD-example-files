# TD-example-files
Some example files and techniques I found along my way

This repository is a collection of TouchDesigner example files focused on understanding how different operators (OPs) work through simple, modular experiments.
Each folder contains an independent project demonstrating a specific technique, workflow, or concept.

This project is intended as a learning resource for:
-beginners starting with TouchDesigner
-artists exploring procedural workflows
-anyone interested in breaking down how specific operators behave

Is organized into folders by topic and complexity. Example structure:
basics/ → fundamental operator usage
intermediate/ → combined techniques and workflows
advanced/ → more complex systems and experiments

You can:
--open and explore the files
--modify and adapt them
--reuse parts in your own projects

These examples are meant to be hands-on learning tools, so experimentation is encouraged.
